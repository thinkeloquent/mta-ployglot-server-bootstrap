# .Agent.md — `fastapi_server` (Python / FastAPI)

> LLM contract for agents working inside this package. Read before editing code, adding addons, or consuming the library. Mirrors `fastify_server/.Agent.md` — behavior must stay parity-equivalent.

---

## 1. What this package is

Runtime-neutral **bootstrap orchestrator** for FastAPI. It discovers user files at boot (`*.env.py`, `*.lifecycle.py`, `*.routes.py`), runs them through a priority-ordered **addon pipeline**, and returns a ready-to-serve `FastAPI` app. It does **not** ship business code.

- Package name on PyPI: `thinkeloquent-fastapi-server`
- Import path: `fastapi_server.bootstrap`
- Python: `>=3.11`
- Parity counterpart: `../fastify_server` (TypeScript/Fastify) — the two are lockstep.

## 2. Contracts (SOURCE OF TRUTH)

The JSON Schemas under `contracts/` are **normative**. All code, examples, and docs must conform.

| Schema | $id | Governs |
| --- | --- | --- |
| `contracts/bootstrap.schema.json` | `https://polyglot.local/bootstrap.schema.json` | Shape of the user-supplied `BootstrapConfig` dict |
| `contracts/hooks.schema.json`     | `https://polyglot.local/hooks.schema.json`     | `onInit` / `onStartup` / `onShutdown` signatures |
| `contracts/loader.schema.json`    | `https://polyglot.local/loader.schema.json`    | `LoaderReport` emitted by every addon |

Parity rule: these three files are **byte-for-byte identical** with `../fastify_server/contracts/`. Enforced by `tests/test_contract_parity.py`. When you change one, copy to the peer and re-run parity tests in the same PR.

### BootstrapConfig (abridged)

```python
{
  "title": str,                    # service name
  "port": int,                     # 0..65535
  "host": str,                     # default "0.0.0.0"
  "profile": str | None,           # free-form env label
  "logger": {"level": "trace|debug|info|warn|error|fatal|silent", "json_format": bool},
  "paths": {
    "environment": str | list[str],
    "lifecycles":  str | list[str],
    "routes":      str | list[str],
    "apps":        str | list[str],
  },
  "initial_state": dict[str, Any],  # deep-cloned into each request
}
```

Paths are **relative to `SetupOptions.base_dir`** (defaults to `os.getcwd()`). Validator: `fastapi_server.bootstrap.contract.validators.validate_bootstrap_config`.

### LoaderReport (every addon returns one)

```python
LoaderReport(
  name: str,
  discovered: int, validated: int, imported: int, registered: int, skipped: int,
  errors: list[LoaderReportError],  # each {step, error, path?}
  details: dict[str, Any],
)
```

Invariants:
- Non-negative counters.
- `errors[].step` is a short verb: `discover` | `import` | `shape` | `register` | `run`.
- An addon **must never raise** out of `run`; catch and append to `report.errors` instead. The registry falls back to an empty report with a single `run` error if an addon leaks an exception.

### Hook signatures

```python
def on_init(server: FastAPI, config: ResolvedBootstrapConfig) -> None | Awaitable[None]: ...
async def on_startup(server, config) -> None: ...
async def on_shutdown(server, config) -> None: ...
```

All three accept `(server, config)` and may be sync or async. Return value is ignored. Hook failures are **logged, not raised**.

## 3. Public API

Import everything from `fastapi_server.bootstrap`:

| Export | Purpose |
| --- | --- |
| `setup(adapter, addons, config, opts=None)` | Builds and returns a ready `FastAPI` app. Async. |
| `SetupOptions(base_dir=None, defaults={}, logger=None)` | Override base dir, defaults, and logger. |
| `create_fastapi_adapter()` | FastAPI + uvicorn runtime adapter. |
| `environment_addon` (priority 10) | Imports `*.env.py`. Side-effect only. |
| `lifecycle_addon` (priority 20) | Registers `on_init` / `on_startup` / `on_shutdown` from `*.lifecycle.py`. |
| `route_addon` (priority 30) | Mounts routers from `*.routes.py`. |
| `Addon(name, priority, run)` | Build custom addons. |
| `Registry` | Container; supports priority ordering and failure isolation. |
| `compose_addons(*groups)` | Flatten groups into a single addon list. |
| `create_loader_report(name)`, `report_error(report, step, err, path=None)` | Report helpers. |
| `merge_config(defaults, user, base_dir) -> ResolvedBootstrapConfig` | Merge + resolve paths. |
| `resolve_paths(paths, base_dir) -> ResolvedPaths` | Resolve `str | list[str]` entries into absolute lists. |
| `sort_by_numeric_prefix(paths)` | Deterministic ordering of discovered files. |
| `validate_bootstrap_config(dict) -> dict` | Raises `BootstrapConfigError`. |

### `setup` in one sentence

`setup` merges defaults+user config (→ `ResolvedBootstrapConfig`), creates the FastAPI app via the adapter, runs every addon in priority order (10 → 20 → 30 → custom), logs per-addon reports, deep-copies `initial_state` into every request, runs init hooks inline, schedules startup/shutdown hooks on the ASGI lifespan, installs graceful shutdown, and returns the app.

## 4. Discovery rules — what the built-in addons look for

Each addon scans the directories listed in `config.paths.<key>` (already resolved to absolute paths).

| Addon | Suffixes matched | Module contract |
| --- | --- | --- |
| `environment_addon` | `.env.py` | **Imported for side effects** (e.g. `os.environ.setdefault(...)`). No exports required. |
| `lifecycle_addon`   | `.lifecycle.py` | Any of `on_init`, `on_startup`, `on_shutdown` callables (sync or async). Modules with none are counted as `skipped`. |
| `route_addon`       | `.routes.py` | Must expose **one** of `mount`, `router`, `default`. A callable gets `(app, config)` and may return an `APIRouter`; a bare router object is registered as-is. |

Files are sorted by **numeric prefix** via `sort_by_numeric_prefix`. Use `01_`, `10_`, `20_` style filenames when ordering matters.

## 5. Usage examples

### 5.1 Minimal quickstart (see `examples/quickstart/`)

```python
import asyncio
import uvicorn
from fastapi_server.bootstrap import (
    setup, SetupOptions, create_fastapi_adapter,
    environment_addon, lifecycle_addon, route_addon,
)

async def build():
    return await setup(
        create_fastapi_adapter(),
        [environment_addon, lifecycle_addon, route_addon],
        {
            "title": "my-service",
            "port": 8000,
            "paths": {
                "environment": ["config/environment"],
                "lifecycles":  ["config/lifecycles"],
                "routes":      ["config/routes"],
            },
        },
        SetupOptions(base_dir="."),
    )

app = asyncio.run(build())
uvicorn.run(app, host="0.0.0.0", port=8000)
```

### 5.2 User files

```python
# config/environment/01_defaults.env.py — side-effect only
import os
os.environ.setdefault("BUILD_ID", "local-dev")

# config/lifecycles/10_startup.lifecycle.py
def on_init(app, config):
    app.state.booted_at = __import__("time").time()

async def on_startup(app, _config): ...
async def on_shutdown(app, _config): ...

# config/routes/10_healthz.routes.py
from fastapi import APIRouter
def mount(app, config) -> APIRouter:
    r = APIRouter()
    @r.get("/healthz")
    async def _(): return {"status": "ok", "service": config.title}
    return r
```

### 5.3 Custom addon (always return a report, never raise)

```python
from fastapi_server.bootstrap import Addon, create_loader_report, report_error

def _run(server, config, ctx):
    report = create_loader_report("metrics")
    try:
        ctx.get_adapter().decorate(server, "_metrics", {"counter": 0})
        report.registered = 1
    except Exception as err:
        report_error(report, "register", err)
    return report

metrics_addon = Addon(name="metrics", priority=40, run=_run)
```

## 6. Invariants an agent must preserve

1. **Contract parity**: never edit `contracts/*.json` without mirroring to `../fastify_server/contracts/` in the same PR.
2. **Addon isolation**: a failing addon **must not** break the others. Catch inside `run`; push to `report.errors`.
3. **Report shape**: every addon returns a `LoaderReport` with all numeric counters ≥ 0 and an `errors` list (possibly empty).
4. **Priority semantics**: environment (10) runs before lifecycle (20) runs before routes (30). User addons should slot in using sparse integers (e.g. 15, 25) and document why.
5. **Deterministic ordering**: always sort discovered files with `sort_by_numeric_prefix` — tests depend on it.
6. **Path resolution is the orchestrator's job**: addons receive *absolute* paths in `config.paths.*`. Do not re-resolve.
7. **Hooks may be sync or async**: always wrap with `inspect.isawaitable(result)` before awaiting.
8. **No top-level I/O on import**: keep module import cheap; real work happens inside `run` / hooks.
9. **Logging is structured**: use `ctx.logger` (`.info .warn .error .debug`), not bare `print`.
10. **Mirror naming**: Python uses `snake_case` (`environment_addon`, `on_startup`); TS uses `camelCase` (`environmentAddon`, `onStartup`). Names differ — *behavior* doesn't.

## 7. Testing contract

- Contract tests live in `tests/` and must pass before merge.
- `tests/test_contract_parity.py` proves the three schemas equal their `../fastify_server/contracts/` peers byte-for-byte.
- `tests/test_contract.py` exercises `merge_config`, `resolve_paths`, `validate_bootstrap_config`, and report helpers.
- `tests/test_registry.py` exercises priority ordering and failure isolation.
- Run: `pytest` (uses `asyncio_mode = "auto"` from `pyproject.toml`).

When adding a new feature: (a) add a test in `tests/`, (b) update this file's §3 or §4 if the public surface changed, (c) mirror to `fastify_server` if it affects behavior or schemas.

## 8. Do / Don't for agents

**Do**
- Keep `fastapi_server.bootstrap` the single public import surface.
- Use `create_loader_report` + `report_error` — never build report dicts by hand.
- Treat `ResolvedBootstrapConfig` as read-only after `merge_config` returns.
- Prefer `ctx.register_*_hook` over poking `app.state` directly.
- Add new discovery suffixes to the addon's `_SUFFIXES` tuple — keep the list short and explicit.

**Don't**
- Don't publish from this repo directly — produce `Makefile.publish` targets and let the user trigger them. (See project memory.)
- Don't introduce cross-package imports from `fastify_server/`.
- Don't rename public exports without updating `__init__.py`, the README, and the parity twin.
- Don't swallow errors silently — they must land in a `LoaderReport.errors` entry.
- Don't add runtime dependencies beyond `fastapi` and `uvicorn` unless discussed.
